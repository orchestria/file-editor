# Text editing tools

Tool collection to read and write text files.

## Usage

Read:

```
$ hatch run read_file.py '{{"path": "/tmp/foo"}}'
```

Write:

```
$ hatch run write_file.py '{{"path": "/tmp/foo", "content": "This is a temporary file"}}'
```
